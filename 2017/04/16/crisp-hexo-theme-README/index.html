crisp-hexo-theme

Preview: Live Demo

A minimalist theme inspired by a Ghost theme of the same name by Kathy Qian.

Installation

Requirements

  • hexo-renderer-scss

Install

Enable

Set theme: crisp in your site _config.yml

Configuration

Primary link color

Modify $themeColor in source/styles/_theme.scss

Theme Settings_config.yml

  • logo - Your website logo
  • dark - true | false toggle the dark version of the theme
  • rss - RSS subscription link (change if using Feedburner)
  • google_analytics - Google Analytics tracking ID

  • disqus - Disqus shortname
